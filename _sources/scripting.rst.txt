Scripting
=========

`mmSolver` can be controlled and extended using Python Scripting.

`mmSolver` hyas an API, as well as tools and utilties that are all
accessable once `mmSolver` is installed. All modules are available
inside the `mmSolver` package namespace.

The Python modules are split into categories, based on the
functionality of the module:
