mmSolver.tools.linkmarkerbundle¶

‘Link / Unlink Marker Bundles’ tool

Tools¶

The ‘Link / Unlink Marker Bundles’ tool.

mmSolver.tools.linkmarkerbundle.tool.link_marker_bundle()¶

Select a marker node, and a bundle node, run to link both nodes.

mmSolver.tools.linkmarkerbundle.tool.unlink_marker_bundle()¶

All selected markers are disconnected from their respective bundle.

Library¶

Library functions to perform ‘Link / Unlink Marker Bundle’ tool functions.

mmSolver.tools.linkmarkerbundle.lib.link_marker_bundle(mkr_node, bnd_node)¶

Try to connect the two marker and bundle nodes.

Constants¶

Contains constant values for the ‘Link / Unlink Marker Bundles’ tool.
