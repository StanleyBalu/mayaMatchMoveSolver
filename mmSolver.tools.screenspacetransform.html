mmSolver.tools.screenspacetransform¶

Convert a 3D transform into a screen-space transform.

Tool¶

Convert a transform into a screen-space transform.

mmSolver.tools.screenspacetransform.tool.main()¶

Screen-Space Bake.

Usage:

  1. Select transform nodes.

  2. Activate viewport.

  3. Run tool.

  4. A new locator is created under the active camera

Library¶

Position locator at the screen-space position.
