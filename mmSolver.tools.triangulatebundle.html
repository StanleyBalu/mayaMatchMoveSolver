mmSolver.tools.triangulatebundle¶

Calculate a 3D position of a bundle, based on the camera and 2D marker.

Tools¶

Triangulate a 3D Bundle from a 2D Marker.

mmSolver.tools.triangulatebundle.tool.main()¶

Triangulate Bundle using camera and Marker.

Usage:

  1. Select markers or bundles (or both).
  2. Run tool.
  3. Bundle is triangulated in TX, TY and TZ.

Library¶

Position Bundle under the Marker.

mmSolver.tools.triangulatebundle.lib.get_marker_frame_list(mkr_node)¶

Get the list of frames that this marker is enabled for.

mmSolver.tools.triangulatebundle.lib.triangulate_bundle(bnd, relock=None)¶

Triangulate a 3D bundle position.

Parameters:
  • bnd (Bundle) – Bundle to be triangulated.
  • relock (bool) – If True any bundle translate attributes will be unlocked, changed then relocked.
